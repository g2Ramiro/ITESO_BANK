ITESO BANK es una plataforma integral diseñada para detectar y prevenir fraude financiero en tiempo real mediante el uso combinado de tecnologías NoSQL como Cassandra, MongoDB y Dgraph. El sistema captura y almacena transacciones históricas, gestiona perfiles documentales de usuarios y modela relaciones complejas entre cuentas, dispositivos e identidades para identificar patrones atípicos y posibles redes fraudulentas. A través de un conjunto de scripts en Python, la solución permite poblar bases de datos, ejecutar consultas operativas y realizar investigaciones avanzadas que integran información temporal, documental y relacional. Este proyecto demuestra cómo múltiples bases de datos distribuidas pueden trabajar juntas para ofrecer un entorno robusto, escalable y eficiente para el análisis antifraude.
